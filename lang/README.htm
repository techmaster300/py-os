PyOS Translation Guide

Adding a new language is simple:

  1. Copy en.json and rename it to your language code (e.g. fr.json, de.json, es.json).
  2. Translate the values (right side of each colon). Never change the keys (left side).
  3. Restart PyOS. Your language appears in Settings → General → Language.

Example

{
  "desktop.header": "PyOS Desktop",
  "desktop.greeting": "Welcome to PyOS..."
}

Becomes:

{
  "desktop.header": "Bureau PyOS",
  "desktop.greeting": "Bienvenue sur PyOS..."
}

Format variables

Some values contain {name} or {count} placeholders. Keep them as-is in your translation.

Example: "Launching {name}" in French → "Lancement de {name}"
